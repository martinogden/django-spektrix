===============
Django-spektrix
===============

Installation
============

1. From source::

    $ git clone git://github.com/cahoona/django-spektrix.git django-spektrix
    $ cd django-spektrix
    $ python setup.py install

2. Using pip (Recommended)::

    $ pip install git+git://github.com/cahoona/django-spektrix.git


Requirements
============

* Django >= 1.2.5
* lxml
