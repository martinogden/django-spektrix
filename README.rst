===============
Django-spektrix
===============

Installation
============

From source:

    $ git clone git://github.com/martinogden/django-spektrix.git django-spektrix
    $ cd django-spektrix
    $ python setup.py install

Using pip (Recommended):

    $ pip install git+git://github.com/martinogden/django-spektrix.git


Requirements
============

* Django >= 1.2.5
* lxml
